# 수능 중국어 기본 어휘 VOCA 입니다.

## 개요

**이 프로그램은 html과 javascript로 동작합니다.**
중국어 단어를 스프레드 시트로 정리하고 그 외, 화면 기본 ui를 만드는 과정에서 _Gemini_ 의 도움이 있었으며
스프레드시트 크롤링 등의 핵심 과정은 자작입니다.
모든 문제는 4지선다 입니다.

**VOCA의 출처는 _2027학년도 EBS 수능특강 중국어 I 부록_ 입니다.**


## 기능설명
현재 이 프로그램은 다음과 같은 문제 출제 방식이 있습니다.
* 한국어 뜻 보고 중국어 한자 맞추기
* 중국어 한자 보고 한국어 뜻 맞추기
* 중국어 병음 보고 한국어 뜻 맞추기
* 중국어 한자 보고 동일 성조 배열 한자 맞추기
* 단어 리스트 보기
